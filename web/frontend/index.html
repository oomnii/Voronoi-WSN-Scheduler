What this project shows

  • Voronoi-based redundancy detection: tiny bounded Voronoi area → redundant → sleep.
  • Backup nodes: OFF nodes are standby for failures.
  • Fault tolerance: random failures + greedy backup activation.
  • Comparison: Voronoi vs random baselines.
  • Experiments: density vs backups and energy saving trends.
